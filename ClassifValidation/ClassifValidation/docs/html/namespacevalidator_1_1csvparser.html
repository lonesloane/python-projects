def validator.csvparser.main ( )

Module entry point.

Loops through all available sub-folders (i.e. corpuses) in outputFolder and compares manual annotations (manualTagsFileName) with annotations coming from Temis (temisTagsFileName)

Temis annotations are generated from xml files using temisxmlparser Manual annotations are copied in the respective corpus folders using manualtagsimport

Remember to update the config.ini file accordingly.
